
Instead of default 'user Profile' directory, 'Chrome User Profile' directory is created to store a saperate user profile for any session/visited web site. This is specially used for zerodha kite login session.
In the project directory, there is a file named as 'Selenium Chrome Session.bat' to run windows command. This file requires two paths 1st one is the chrome.exe file and 2nd one is the path of this directory(Chrome User Profile).
